\section{System Definition}
\label{sec:subSysDef}
Based on the \nameref{chap:systemDef} and \nameref{sec:requirements} we define the subsystem we are implementing as: 
\begin{center}
\framebox[0.85\textwidth][c]{
	\parbox{0.8\textwidth}{
		\textsl{
			A subsystem of \system{} that implements project groups in Moodle and allow for administration and usage thereof. 
			The subsystem includes a virtual meeting place, which integrates the other subsystems. 
		}
	}
}
\end{center}
The subsystem must conform to the overall system definition, defined in \chapref{chap:systemDef}.
